Stock Returns Determinants SPROJ

Overview of datasets and objectives.

Project Objectives

Understand the drivers of stock returns using various technical, macroeconomic, and fundamental indicators.

Daily Stock Returns

Analysis based on technical indicators and machine learning models.

ML Model Comparison

CAPM, GBM, Candlestick Patterns, and Combined Technical Indicators.

Monthly Stock Returns

Evaluating macroeconomic indicators using pooled OLS crossection analysis.

Key Variables

Crude Oil, Natural Gas, and Gold.

Cross-Section Analysis

Fundamental indicators such as Revenue, Full-Time Employees, and P/E Ratio.

Event Studies

Investigation of key events and their impact on stock market performance.
